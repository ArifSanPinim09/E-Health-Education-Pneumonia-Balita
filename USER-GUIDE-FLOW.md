```
┌─────────────────────────────────────────────────────────────┐
│                    User Login Pertama Kali                   │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│              Redirect ke Dashboard Page                      │
└─────────────────────────────────────────────────────────────┘
                              ↓

                              ↓

│              Data Loaded Successfully                        │

                              ↓

                              ↓
                    ┌─────────┴─────────┐

                    │                   │
        ┌───────────┘                   └───────────┐
        ↓                                           ↓
┌──────────────────┐                    ┌──────────────────────┐
```

## 🎯 Element Highlighting Flow

```

└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────┘
                         ↓

└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────┘
                         ↓

└─────────────────────────────────────────────────────────┘
                         ↓
                ┌────────┴────────┐
                │                 │
              Next              Close
                │                 │
                ↓                 ↓
        Go to Next Step    Destroy Guide
```

## 💾 LocalStorage Flow

```
┌─────────────────────────────────────────────────────────┐
│              Check localStorage                          │
│  localStorage.getItem('guide_seen')                      │
└─────────────────────────────────────────────────────────┘
                         ↓
                ┌────────┴────────┐
                │                 │
              null            'true'
                │                 │
                ↓                 ↓
        ┌──────────────┐  ┌──────────────┐
        │ First Time   │  │ Already Seen │
        │ Show Guide   │  │ Skip Guide   │
        └──────────────┘  └──────────────┘
                │
                ↓
        ┌──────────────┐
        │ User Complete│
        │ or Close     │
        └──────────────┘
                ↓
        ┌──────────────────────────────┐
        │ localStorage.setItem(        │
        │   'guide_seen', 'true'       │
        │ )                            │
        └──────────────────────────────┘
                ↓
        ┌──────────────┐
        │ Next Visit   │
        │ → Skip Guide │
        └──────────────┘
```

## 🎨 Component Integration

```
app/(user)/dashboard/page.tsx
        ↓
┌──────────────────────────────────────┐
│  Import: startUserGuide()            │
│  Import: shouldShowGuide()           │
│  Import: driver-custom.css           │
└──────────────────────────────────────┘
        ↓
┌──────────────────────────────────────┐
│  useEffect Hook                      │
│  - Check loading state               │
│  - Check profile & progress          │
│  - Call shouldShowGuide()            │

        ↓
┌──────────────────────────────────────┐
│  setTimeout(startUserGuide, 800)     │
└──────────────────────────────────────┘
        ↓
┌──────────────────────────────────────┐
│  lib/userGuide.ts                    │
│  - Create driver instance            │
│  - Define steps                      │
│  - Start guide                       │
└──────────────────────────────────────┘
        ↓
┌──────────────────────────────────────┐
│  Driver.js Library                   │
```
